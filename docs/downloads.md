# Download Materials

You can download the course materials for offline use. These packages include all lessons and exercises in PDF or DOCX format.

The latest materials are available from the GitHub [release](https://github.com/digitalpalidictionary/dpd-pali-courses/releases/latest).

## Offline Website

Download the entire website for offline use on a laptop or desktop computer.

- [Offline Website (ZIP)](https://github.com/digitalpalidictionary/dpd-pali-courses/releases/latest/download/website_offline.zip)

**How to use:** Download the ZIP, unzip it, then open `index.html` in any browser (Chrome, Firefox, Safari).

> Note: The search bar will not work when browsing locally. All course content and navigation work normally.
