# Adverbs of time

They are part of the group of indeclinable words in Pāli.
***
Adverbs of time describe **when** the action is done.
***
They often come **first** in the sentence.
***
| | |
|---|---|
| pubbe     | before, previously                       |
| āyatiṃ    | in future                                |
| dāni      | now                                      |
| idāni     | now                                      |
| yadā      | when, whenever                           |
| pacchā    | afterwards                               |
| ajja      | today                                    |
| tadā      | then                                     |
| sadā      | always                                   |
| sāyaṃ     | in the evening; at night                 |
| kadā      | when                                     |
| pāto      | in the morning                           |
| ekadā     | one day; sometimes                       |
| suve      | tomorrow                                 |
| purā      | formerly, earlier                        |
| atippago  | too early                                |
| aciraṃ    | recently, soon                           |
| ciraṃ     | for a long time                          |
| atisāyaṃ  | too late (in the day); too late at night |
| kālena    | at the proper time                       |
***
