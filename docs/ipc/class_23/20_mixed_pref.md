**Mixed Prefixes**

|---|---|---|---|---|
| --- | --- | --- | --- | --- |
| abhi + ā | √cikkh | to show | abbhācikkhati | misrepresents |
| abhi + ni | √sad | to sink | abhinisīdati | sit down hard |
| abhi + pa | √atth | to wish | abhipatthayati | longs for |
| adhi + ā | √vas | to live | ajjhāvasati | settles |
| adhi + ava | √kir | to scatter | ajjhokirati | pours down |
| anu + pa | √ikkh | to look | anupekkhati | inspects |
| anu + pari | √vatt | to go on | anuparivattati | circles around |
| anu + vi | √lok | to see | anuviloketi | looks around |
| pari + nī | √vā | to blow | parinibbāyati | is extinguished |
| pati + ā | √gam | to go | paccāgacchati | returns |
| pati + anu | √bhū | to be | paccanubhavati | enjoys, undergoes |
| pati + saṃ | √har | to carry | paṭisaṃharati | withdraws |
| pati + saṃ | √vid | to sense | paṭisaṃvedayati | experiences |
| pati + vi | √gam | to go | paṭivigacchati | disappears |
| saṃ + adhi | √gam | to go | samadhigacchati | attains |
| saṃ + anu | √dis | to see | samanupassati | considers as |
| saṃ + pa | √yuj | to join | sampayojeti | quarrels |
| sam + pati | √is | to wish | sampaṭicchati | accepts, receives |
| saṃ + ud | √chid | to cut | samucchindati | eradicates |
| upa + ni | √band | to tie | upanibandhati | ties down onto |
| upa + pari | √ikkh | to see | upaparikkhati | investigates |
| upa + saṃ | √kam | to go | upasaṅkamati | approaches |
| vi + ā | √kar | to do | byākaroti | answers |
| vi + nī | √bhuj | to bend | vinibbhujati | unrolls |

***
It is important to note that the prefixes are used not only with verbs but also with participles, verbal derivatives, nouns and adjectives.
