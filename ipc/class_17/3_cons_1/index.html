    Possibility #1

    When a word or particle ends in a vowel,

    the following consonant is generally reduplicated.

    There are thousands of such examples.

    e.g. pañca + khanda > pañcakkhandha

    In Kaccāyana this is called (28) para dvebhāvo ṭhāne and

    (29) vagge ghosāghosānaṃ tatiyapaṭhamā.

    vowel + l | | | | | |---|---|---|---| | vijju | latā | vijjullatā | fem. forks of lightning | | ava > o | lujjati | ullujjati | pr. is breaking down |

    vowel + s | | | | | |---|---|---|---| | pa | sasati | passasati | pr. exhales, breathes out | | diṭṭha | suta | diṭṭhassuta | nt. what is seen and heard | | anu | saritar | anussaritar | masc. who recollects |

    Counter-examples:

    ajāta + sattu > ajātasattu

    upa + pajjati > upapajjati

    pa + dāleti > padāleti
