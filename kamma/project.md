# Project Guide: DPD Pāḷi Courses

## What it is and why
The primary purpose of this project is to maintain and deliver Pāḷi course materials—specifically the Beginner Pāḷi Course (BPC) and Intermediate Pāḷi Course (IPC)—via a static website and high-quality generated documents (PDF and DOCX). It ensures that original materials from Google Docs are preserved in a modern, searchable, and version-controlled Markdown format.

## Who it is for
The target audience includes:
- Students of the Digital Pāḷi Dictionary (DPD) Beginner and Intermediate Pāḷi Courses.
- General practitioners interested in learning Pāḷi for personal study of the Tipiṭaka.

## One-off or ongoing
This is an **ongoing effort**. The focus is on maintaining the materials, fixing errors, and continuously updating the generation scripts and website to ensure they remain functional and accurate.

## What it will produce
- A searchable static website built with MkDocs.
- High-quality PDF volumes generated via WeasyPrint.
- Editable DOCX documents generated via Pandoc.
- Automated verification reports to ensure content integrity across all formats.

## How we'll know it worked
Success is defined by:
- Materials being fully converted to Markdown with high-quality website, PDF, and DOCX versions automatically generated and kept in sync.
- Students easily accessing and using materials for Pāḷi studies across multiple formats without data loss or formatting errors.
- A streamlined maintenance workflow that allows for easy updates and robust verification of content integrity.
